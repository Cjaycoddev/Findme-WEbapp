The application uses React for front-end, Django for backend and MySQL for database.
I used vs code to write and develop the code.
To run the application they have to be installed.

1. Open cmd, navigate to the folder "Findme Application" by running cd "Findme Application" and then type  Npm start  to run react.

2. Open vs code and in the terminal navigate to virtual environment folder "djangoenv" and activate it buy running  .\djangoenv\Scripts\Activate.ps1  

3. Navigate to project folder "FindMeWebapp" by running cd .\FindMeWebapp\

4.Start Django server by running    python manage.py runserver

5.Use React's development server that runs at the address
 http://localhost:3000.
